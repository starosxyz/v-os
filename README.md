# v-os
V-OS userspace virtual operating system kernel. it support linux/macOS/Windows.
## Quick Start
    # clone v-os
    mkdir -p /data/v-os
    git clone https://github.com/starosxyz/v-os.git /data/v-os
    
    #compile
    cd /data/v-os/build
    source .env.sh
    make debug=0 staitc=0 os=ubuntu20
